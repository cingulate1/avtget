Avtget Sender

This extension sends the current tab (or a right-clicked link) to the Avtget desktop app over a local HTTP connection. The values below must match the HTTP server settings in Avtget's config.ini (or its Settings dialog).

Connection

Change the port if you customized http_server_port.
Copy the http_server_token value from Avtget's config.ini. The token is auto-generated on Avtget's first launch and never changes unless you clear it.
